# Vaidya Solutions

We build **Gideon**, a private, proprietary legal AI platform.

Our production work is TypeScript-first, with Go used for selected stateless, Lambda-oriented, and performance-sensitive services. We also maintain selected public open-source projects and libraries.

## Principles

- **Reliability**: graceful degradation, bounded retries, and chaos-ready design.
- **Clear boundaries**: strong interfaces between UI, agents, MCP tooling, and infrastructure.
- **Observability**: structured logs and traces from day one.
- **Privacy-by-default**: no secrets, credentials, or customer data in logs, commits, or public issues.
- **Security hygiene**: secret scanning, SBOMs, least privilege, and hardened CI.

## Contributing

Focused issues and pull requests are welcome only in public repositories that explicitly accept them.

Private and proprietary repositories — including Gideon — are not open to unsolicited external contribution. External project work occurs only through an approved, scoped engagement.

## Security

Report suspected vulnerabilities privately to **jay@vaidyasolutions.com**. Do not open a public issue for sensitive reports.
